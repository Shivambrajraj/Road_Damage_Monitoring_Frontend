# RoadAnomalyAI — Automated Road Anomaly Detection Platform

An automated, data-dense infrastructure inspection system that integrates computer vision analytics with geographic information systems (GIS) to track, classify, and register municipal road surface damage in real time.

##  Tech Stack Matrix
* **Frontend Core:** React.js (Functional architecture with modern custom hooks)
* **Styling Framework:** Tailwind CSS v4 (Design tokens system)
* **Geospatial Mapping Engine:** Leaflet.js & React-Leaflet
* **Data Ingestion Client:** Axios (Unified request/response interception filters)

##  Key Feature Slices
1. **Central Control Console:** Dynamic table indices displaying anomaly catalogs alongside localized search and severity prioritization filter matrices.
2. **Launch Analytics Run:** Interactive multipart form payload upload pipeline supporting device GPS telemetry syncing via the browser's Geolocation API.
3. **Geospatial GIS Canvas:** Interactive high-contrast dark-matter tile canvas mapping defect coordinates into active inspectable pinpoint nodes.
4. **Predictive Analytics Hub:** Volumetric density tracking widgets outlining long-term infrastructure degradation velocities over historical timelines.
